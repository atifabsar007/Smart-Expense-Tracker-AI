# 💰 Smart Expense Tracker AI

A C++ based intelligent expense tracking system that helps users manage their daily finances, analyze spending habits, and receive smart budget suggestions.

---

## 🚀 Features

- Add income and expenses
- Category-based expense tracking (Food, Travel, Study, etc.)
- Automatic financial report generation
- Overspending detection system
- Smart budget suggestions (rule-based AI)
- Undo last transaction feature
- File-based persistent storage

---

## 🧠 Concepts Used

- Object-Oriented Programming (OOP)
- STL (vector, map, stack)
- File Handling
- Basic AI rule-based logic
- Data analysis techniques

---

## 📊 Example Insight

- If Food > 40% of total spending → Suggest cooking at home
- If Travel increases suddenly → Suggest planning budget routes

---

## 📁 Project Structure

- main.cpp → Entry point
- tracker.h / tracker.cpp → Core logic
- expense.h → Expense model
- fileio.cpp → Data storage system
- analytics.cpp → Smart insights engine
